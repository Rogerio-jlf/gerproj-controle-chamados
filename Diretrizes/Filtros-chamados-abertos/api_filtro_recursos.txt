             ┌──────────────────────────────┐
             │ Recebe a requisição GET      │
             └────────────┬─────────────────┘
                          │
                          ▼
          ┌───────────────────────────────────────────────┐
          │ Extrai parâmetros da URL:                     │
          │ - anoParam                                    │
          │ - mesParam                                    │
          │ - isAdmin                                     │
          │ - codRecurso (se não for admin)               │
          │ - cliente (se for admin e quiser filtrar)     │
          └────────────┬──────────────────────────────────┘
                       │
                       ▼
         ┌────────────────────────────────────────────────┐
         │ Valida os parâmetros                           │
         │ - mes entre 1 e 12                              │
         │ - ano entre 2000 e 3000                         │
         │ - se não for admin → codRecurso obrigatório     │
         └────────────┬────────────────────────────────────┘
                      │
                      ▼
       ┌──────────────────────────────────────┐
       │ Cria intervalo de datas do mês       │
       │ dataInicio: 01/mes/ano 00:00         │
       │ dataFim: 01/mes+1/ano 00:00          │
       └────────────┬─────────────────────────┘
                    │
                    ▼
      ┌────────────────────────────────────────────────────────────┐
      │ Cria objeto de filtro para consulta no banco               │
      │ - data_chamado entre dataInicio e dataFim                  │
      │ - se for admin e cliente foi informado → filtra por cliente│
      │ - se não for admin → filtra por codRecurso                 │
      └────────────┬───────────────────────────────────────────────┘
                   │
                   ▼
     ┌──────────────────────────────────────────────────────────────┐
     │ Consulta no banco de dados usando Prisma ORM:                │
     │ - Busca chamados com os filtros definidos                    │
     │ - Agrupa por recurso (distinct cod_recurso)                  │
     │ - Pega apenas o nome do recurso relacionado a cada chamado   │
     └────────────┬─────────────────────────────────────────────────┘
                  │
                  ▼
   ┌───────────────────────────────────────────────────────────┐
   │ Transforma o resultado:                                   │
   │ - Extrai apenas os nomes dos recursos                     │
   │ - Remove nulos e nomes em branco                          │
   │ - Ordena em ordem alfabética (localeCompare, pt-BR)       │
   └────────────┬──────────────────────────────────────────────┘
                │
                ▼
     ┌────────────────────────────────────┐
     │ Retorna os nomes dos recursos      │
     │ em formato JSON para o frontend    │
     └────────────────────────────────────┘
