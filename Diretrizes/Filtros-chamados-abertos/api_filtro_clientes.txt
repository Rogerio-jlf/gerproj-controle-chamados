             ┌──────────────────────────────┐
             │ Recebe a requisição GET      │
             └────────────┬─────────────────┘
                          │
                          ▼
          ┌────────────────────────────────────┐
          │ Extrai parâmetros da URL:          │
          │ - anoParam                         │
          │ - mesParam                         │
          │ - isAdmin                          │
          │ - codRecurso (se não for admin)    │
          └────────────┬───────────────────────┘
                       │
                       ▼
         ┌───────────────────────────────────────────┐
         │ Valida os parâmetros                      │
         │ - mes entre 1 e 12                         │
         │ - ano entre 2000 e 3000                   │
         │ - se não for admin → codRecurso obrigatório│
         └────────────┬──────────────────────────────┘
                      │
                      ▼
       ┌──────────────────────────────────────┐
       │ Cria intervalo de datas do mês       │
       │ dataInicio: 01/mes/ano 00:00         │
       │ dataFim: 01/mes+1/ano 00:00          │
       └────────────┬─────────────────────────┘
                    │
                    ▼
      ┌──────────────────────────────────────────┐
      │ Cria objeto de filtro para o banco       │
      │ - data_chamado entre dataInicio e dataFim│
      │ - se não for admin, filtra por codRecurso│
      └────────────┬─────────────────────────────┘
                   │
                   ▼
     ┌──────────────────────────────────────────────────┐
     │ Consulta no banco com Prisma ORM:                │
     │ - Busca chamados no intervalo de datas           │
     │ - Filtra por recurso se necessário               │
     │ - Agrupa por cliente (distinct cod_cliente)      │
     │ - Pega apenas o nome do cliente                  │
     └────────────┬─────────────────────────────────────┘
                  │
                  ▼
